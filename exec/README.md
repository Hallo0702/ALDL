## 빌드 실행 방법

#### Frontend

client 폴더 안에서 아래의 명령어를 실행한다.

```
환경변수 설정 (.env)

NEXT_PUBLIC_API_URI=http://j7c207.p.ssafy.io:8088
NEXT_PUBLIC_BLOCKCHAIN_URI=http://43.200.253.174:3000
```

```
패키지 설치
$ npm install
```

```
프로젝트 빌드
$ npm run build
```

#### Backend

##### Spring

backend-java 폴더 안에서 아래의 명령어를 실행한다.

```
jar 파일 빌드
$ ./gradlew build
```

```
빌드 파일 실행
$ java -jar aldl.jar
```
